Banner personalizado para Axew :D hecho por mi y por entretenimiento

# Instalacion 
```
apt update && apt upgrade -y
pkg install git -y
git clone https://github.com/Tweaky349/BannerAxew
cd BannerAxew 
chmod +x *
```

# Menu
* Personalizar
`./start --banner`

* Eliminar banner
`./start --delete`

* Discord
`./start --discord`

* Mostrar banner
`./start --show`
